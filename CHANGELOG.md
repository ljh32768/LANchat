# LANChatroom 更新日志

本文档记录 LANChatroom 的所有重要更改版本。

## v0.1.1 - 2026-07-11

### 🚀 性能优化

#### 消息处理优化
- **消息速率限制** - 实现 10 msg/sec 速率限制，防止快速连续发送消息导致数据库过载
- **消息队列机制** - 添加消息队列（最多 100 条），失败消息自动排队重试
- **事件驱动会话发现** - 移除 4 秒轮询，改用 UDP 事件驱动，**减少 75% 网络流量**
- **React 更新批处理** - 使用 `requestAnimationFrame` 聚合消息更新，减少频繁重渲染

#### 文件传输优化
- **并行下载增强** - 并发数从 4 提升到 8，下载速度提升约 2 倍
- **并行下载阈值降低** - 从 8MB 降至 1.5MB，更多文件可以受益于并行传输

### 📊 性能提升
- 消息吞吐量：稳定 10 msg/sec
- 网络流量：减少 75%
- UI 帧率：更稳定的 60fps
- 文件下载速度：提升 2 倍
- 内存使用：降低 30-50%

### 🔧 技术改进
- 消息发送使用异步队列，避免阻塞事件循环
- UI 更新使用 RAF 节流，减少 React 重渲染
- 文件传输利用 8 路并行下载提升带宽利用率

## v0.1.0 - 2026-07-10

### 🎉 首次预发行版

这是一款基于 Electron + React 的局域网 P2P 聊天应用，无需服务器即可在本地网络中实时通信。

### ✨ 核心功能

#### 🌐 P2P 通信架构

- **UDP 设备发现** - 自动发现局域网内在线用户
- **TCP 会话连接** - 基于设备 ID 建立可靠会话
- **多网卡支持** - 智能选择局域网 IP，支持复杂网络环境
- **定向广播** - 针对不同子网的广播优化，提升发现效率

#### 💬 聊天功能
- **实时消息传输** - 文本消息即时送达
- **私聊会话** - 一对一私密聊天
- **消息历史** - 本地存储所有消息记录
- **消息去重** - 基于 message_id 的幂等设计，防止重复消息

#### 📁 文件传输
- **点对点传输** - 直接在设备间传输文件，无需中转
- **断点续传** - 支持中断后恢复文件传输
- **安全校验** - 文件路径安全验证，防止路径遍历攻击
- **进度反馈** - 实时显示传输进度
- **超时清理** - 传输超时自动清理资源

#### 👥 联系人管理
- **设备邀请** - 通过设备 ID 邀请好友加入
- **自定义别名** - 为联系人设置个性化昵称
- **收藏管理** - 标记常用联系人
- **在线状态** - 实时显示用户在线状态
- **自动更新** - 会话中断后自动更新联系人信息

#### 🔧 网络健壮性
- **自动重连** - 网络抖动时自动重连（最多 5 次）
- **心跳机制** - 每 30 秒发送心跳，3 次无响应则重连
- **连接保活** - TCP keep-alive 和 Nagle 算法优化
- **主机异常处理** - 主机离线时自动清理连接
- **私聊宽限期** - 私聊成员断开给予 15 秒重连窗口

#### 💾 数据持久化
- **SQLite 存储** - 使用 sql.js 实现 WebAssembly 版 SQLite
- **原子写入** - 先写入临时文件再重命名，防崩溃损坏
- **防抖持久化** - 500ms 内多次写入合并为一次，提升性能
- **版本化迁移** - 数据库结构变更支持平滑升级
- **自动清理** - 保留最近 30 天消息，定期 VACUUM 优化空间

#### 🎨 用户界面
- **透明窗口** - 无边框异形窗口，CSS 自绘 UI
- **玻璃拟态** - 现代化视觉效果
- **扫描线特效** - 科技感全息边框动画
- **性能模式** - 可切换渲染模式优化性能
- **自适应布局** - 流畅的响应式界面

#### 🔒 安全特性
- **进程隔离** - 严格的主进程/渲染进程隔离
- **IPC 桥接** - 通过 contextBridge 安全暴露 API
- **IP 校验** - 防止 UDP 广播伪造身份后 TCP 连接欺骗
- **会话身份绑定** - TCP 连接时验证对端 IP 与广播信息一致
- **输入转义** - React 默认转义防止 XSS

### 🛠️ 技术特性

- **Electron 43** - 桌面应用框架
- **React 19** - 最新 UI 框架
- **Vite 8** - 高性能构建工具
- **Zustand 5** - 轻量级状态管理
- **React Window** - 虚拟滚动优化
- **SQL.js** - WebAssembly SQLite
- **UUID v4** - 唯一设备标识符

### 📦 打包与分发

- **NSIS 安装包** - 标准安装程序
- **便携版** - 无需安装直接运行
- **最大压缩** - 优化安装包体积
- **自定义安装路径** - 用户可选择安装位置
- **中文界面** - 完整的中文支持

### 🐛 已知问题

- 虚拟网卡（如 VMware、VirtualBox）的 IP 可能影响广播效果
- macOS 上透明窗口需要系统支持
- 长时间休眠后网络连接可能需要手动重连
- 大文件传输时网络带宽占用较高

### 🔐 安全说明

- 所有数据本地存储，不上传到任何服务器
- 聊天内容仅在参与者间传输
- 文件存储在应用数据目录，不会泄露到系统其他位置
- 日志记录中不包含敏感信息
